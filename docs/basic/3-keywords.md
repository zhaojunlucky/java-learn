# Java Keywords

## Access Control

| Access Levels Modifier | Class | Package | SubClass                                    | World |
| ---------------------- | ----- | ------- | ------------------------------------------- | ----- |
| private                | Y     | Y       | N                                           | N     |
| protected              | Y     | Y       | Y                                           | Y     |
| public                 | Y     | Y       | Y                                           | Y     |
| (**default**)          | Y     | Y       | N - different package<br />Y - same package | N     |
